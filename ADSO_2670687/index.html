Concepto de Git

Git es un sistema de control de versiones con el que se puede clonar y guardar repositorios de proyectos, tambien lleva un registro de todos los cambion hechos

(con ul se crea una lista no ordenada (viñetas) y con ol se crea una lista ordenada (numeros), siempre necesario las li

PRINCIPALES COMANDOS DE GIT

  • Git init:Este comando funciona para inicializar un proyecto.
  • Git add .:Permite agregar cambios al area de reparacion.
  • Git commit -m:Permite dejar un comentario dentro de Git.
  • Git status:Permite visualizar el estado actual del archivo.
  • Git log:Permite ver el historial de lo realizado.
  • Git config --global user.name:Este comando permite registrarse con un nombre.
  • Git config --global user.email:este comando permite registrar el email.
  • Git branch:Permite visualizar en que rama nos encontramos.
  • Git -b branch "NAME":Permite cambiar el nombre de la rama.
  • Git remote add origin "URL":Este comando permite entrelazar Git con GitHub.
  • Git config user.name:Este comando sirve para visualizar el nombre del usuario actual.
  • Git config user.email:Permite visualizar el email actual del usuario.
  • Git push:Permite subir el archivo a GitHub.
  • Git checkout -b "nombre de la rama":sirve para visualizar el nombre actual de la rama.
